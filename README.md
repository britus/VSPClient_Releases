## Virtual Serial Port Controller App

All previous releases of the Virtual Serial Port Controller app for macOS are listed here. Old 1.6.x releases are built with QT Framework version 5.15.2. Intel platform. Newer versions for macOS Sequoia and newer with the Apple silicon chips on the ARM64 architecture can be downloaded from Apple's [macOS App Store](https://apps.apple.com/us/app/virtual-serial-port-controller/id6742438113).

### Older Releases
This repository contains macOS x86_64 and Silicon based releases.
